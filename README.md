In the conde environment, do the following:

```

conda activate <environment name>

pip install mne
pip install yacs wandb
```

Major dependency packages:

```
python==3.11.3
torch==2.0.0
mne==1.3.1
yacs==0.1.8
wandb==0.15.2
```
